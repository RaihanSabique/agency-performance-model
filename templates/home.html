Project Description

  • Build REST api that provides:

      Parameters: agency_id, year, date, state

    1. Detailed information using different parameters
    2. Summarized information using different parameters
    3. Graphical analysis information using different parameters
    4. predictive analysis for categorization
  • Build a web app that consumes the API and show charts, tables, prediction results etc
